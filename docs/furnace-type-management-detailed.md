## ভূমিকা

**চুল্লির ধরন ব্যবস্থাপনা** হলো ইট ভাটা ম্যানেজমেন্ট সফটওয়্যারের একটি মৌলিক এবং গুরুত্বপূর্ণ মডিউল। এই মডিউলের মাধ্যমে আপনি আপনার ইট ভাটায় বিদ্যমান বিভিন্ন ধরনের চুল্লি বা ভাটির শ্রেণিবিভাগ তৈরি, সম্পাদনা এবং পরিচালনা করতে পারবেন।

### কী কী কাজ করা যাবে?

✅ নতুন চুল্লির ধরন তৈরি করা  
✅ বিদ্যমান চুল্লির ধরন সম্পাদনা করা  
✅ চুল্লির ধরন সক্রিয়/নিষ্ক্রিয় করা  
✅ চুল্লির ধরনের তালিকা দেখা ও পরিচালনা করা  
✅ প্রতিটি ধরনের অধীনে কতটি চুল্লি আছে তা ট্র্যাক করা  
✅ সার্চ ও ফিল্টারের মাধ্যমে দ্রুত খুঁজে বের করা

## কেন চুল্লির ধরন প্রয়োজন?

চুল্লির ধরন তৈরি করা জরুরি কারণ:

1. **শ্রেণিবিভাগ**: বিভিন্ন ধরনের চুল্লি (যেমন: হফম্যান, জিগজ্যাগ, টানেল) আলাদাভাবে শনাক্ত করা যায়
2. **উৎপাদন বিশ্লেষণ**: কোন ধরনের চুল্লিতে কত উৎপাদন হয় তা বিশ্লেষণ করা যায়
3. **খরচ তুলনা**: বিভিন্ন চুল্লির ধরনের খরচ তুলনা করা যায়
4. **রিপোর্টিং**: চুল্লির ধরন অনুযায়ী রিপোর্ট তৈরি করা যায়
5. **ড্রপডাউন ম্যানেজমেন্ট**: নতুন চুল্লি তৈরির সময় ধরন নির্বাচন করা যায়

> 💡 **গুরুত্বপূর্ণ**: চুল্লির ধরন তৈরি না করলে আপনি নতুন চুল্লি তৈরি করতে পারবেন না। তাই সফটওয়্যার ব্যবহার শুরু করার আগে অন্তত একটি চুল্লির ধরন তৈরি করুন।

## নতুন চুল্লির ধরন তৈরি - ধাপে ধাপে

- HOFF (Hoffman এর জন্য)
- ZIGZ (Zigzag এর জন্য)

- SEMI (Semi-Automatic এর জন্য)
- AUTO (Automatic এর জন্য)

**ভ্যালিডেশন**:
- অবশ্যই লিখতে হবে

- একই কোড দুবার ব্যবহার করা যাবে না
- ছোট হাতের অক্ষর লিখলে অটোমেটিক বড় হাতের অক্ষরে পরিবর্তন হবে

**টিপস**: কোডটি রিপোর্টে ও ইনভয়েসে ব্যবহৃত হবে, তাই সহজ ও মনে রাখার মতো করুন।

## সম্পাদনা (Edit) করা

### কখন সম্পাদনা করবেন?

- শর্ট কোড ঠিক করতে চাইলে

- বিবরণ বা নোট যোগ/পরিবর্তন করতে চাইলে

## সক্রিয়/নিষ্ক্রিয় করা

### কেন সক্রিয়/নিষ্ক্রিয় করবেন?

**সক্রিয় করবেন যখন**:
- ✅ নতুন চুল্লির ধরন ব্যবহারে আনতে চান
- ✅ ড্রপডাউনে দেখাতে চান

**নিষ্ক্রিয় করবেন যখন**:
- ❌ এই ধরন আর ব্যবহার করবেন না
- ❌ সাময়িকভাবে বন্ধ রাখতে চান
- ❌ পুরানো ধরন যা আর প্রযোজ্য নয়

---

### পদ্ধতি ১: তালিকা থেকে দ্রুত টগল

1. তালিকায় যেই চুল্লির ধরনটি টগল করতে চান
2. সেই সারির **⋯** মেনুতে ক্লিক করুন
3. **⚡ Deactivate** অথবা **⚡ Activate** সিলেক্ট করুন
4. সাথে সাথে স্ট্যাটাস পরিবর্তন হবে
5. সবুজ বার্তা দেখাবে:
   ```
   ✓ Furnace type deactivated
   ```
   অথবা
   ```
   ✓ Furnace type activated
   ```

---

### পদ্ধতি ২: এডিট ডায়ালগ থেকে

1. **⋯** মেনু → **✏️ Edit** এ ক্লিক করুন
2. এডিট ডায়ালগে **Active Status** টগল খুঁজুন

```
┌─────────────────────────────────────┐
│ Active Status                       │
│                                     │
│  Inactive  [====●====]  Active ✓   │
│                                     │
└─────────────────────────────────────┘
```

3. টগল সুইচে ক্লিক করে স্ট্যাটাস পরিবর্তন করুন
4. **[আপডেট]** বাটনে ক্লিক করুন

---

### সক্রিয়/নিষ্ক্রিয় করার প্রভাব

#### সক্রিয় থাকলে:
✅ চুল্লি তালিকায় ড্রপডাউনে দেখা যাবে  
✅ নতুন চুল্লি তৈরির সময় নির্বাচন করা যাবে  
✅ রিপোর্টে অন্তর্ভুক্ত হবে  
✅ পরিসংখ্যানে গণনা হবে  

#### নিষ্ক্রিয় করলে:
❌ ড্রপডাউনে আর দেখা যাবে না  
❌ নতুন চুল্লি তৈরির সময় নির্বাচন করা যাবে না  
❌ কিন্তু **বিদ্যমান চুল্লিগুলো অবিকৃত থাকবে**  
❌ শুধু পরিসংখ্যানে "নিষ্ক্রিয়" হিসেবে গণনা হবে  

---

### ⚠️ গুরুত্বপূর্ণ সতর্কতা

**নিষ্ক্রিয় করলে কী হবে না**:
- ❌ বিদ্যমান চুল্লি মুছে যাবে না
- ❌ চুল্লির ডেটা হারাবে না
- ❌ রিপোর্ট থেকে মুছে যাবে না

**শুধু কী হবে**:
- ✅ নতুন চুল্লি তৈরির সময় এই ধরন আর দেখা যাবে না

**উদাহরণ**:
```
আপনার আছে:
- হফম্যান (৩টি চুল্লি সহ) → নিষ্ক্রিয় করলেন

ফলাফল:
- ৩টি চুল্লি অবিকৃত থাকবে ✅
- নতুন চুল্লি তৈরির সময় "হফম্যান" আর দেখা যাবে না ❌
```

## মুছে ফেলা

### ⚠️ সতর্কবার্তা

**বর্তমানে চুল্লির ধরন মুছে ফেলার সুবিধা নেই!**

এটি ইচ্ছাকৃতভাবে করা হয়েছে কারণ:

1. **ডেটা ইন্টেগ্রিটি**: চুল্লির ধরন মুছে ফেললে সম্পর্কিত চুল্লিগুলোর ডেটা সমস্যায় পড়বে
2. **রিপোর্টিং**: পুরানো রিপোর্টে চুল্লির ধরন দেখাবে না
3. **অডিট ট্রেইল**: হিসাব-নিকাশের জন্য পুরানো ডেটা প্রয়োজন

---

### বিকল্প সমাধান: নিষ্ক্রিয় করুন

মুছে ফেলার পরিবর্তে **নিষ্ক্রিয়** করুন:

1. তালিকায় চুল্লির ধরন খুঁজুন
2. **⋯** মেনু → **⚡ Deactivate** এ ক্লিক করুন
3. চুল্লির ধরনটি নিষ্ক্রিয় হয়ে যাবে
4. আর ড্রপডাউনে দেখা যাবে না

---

### ভবিষ্যতে মুছে ফেলার নিয়ম (যদি যোগ করা হয়)

**শর্ত**: শুধু তখনই মুছে ফেলা যাবে যখন:
- ✅ চুল্লির ধরনের অধীনে **০টি চুল্লি** থাকবে
- ✅ কোনো ব্যাচ বা উৎপাদনে ব্যবহৃত হয়নি

**ধাপ**:
1. **⋯** মেনু → **🗑️ Delete** এ ক্লিক করুন
2. কনফার্মেশন ডায়ালগ খুলবে:
   ```
   ┌─────────────────────────────────────┐
   │ 🗑️ Delete Furnace Type             │
   ├─────────────────────────────────────┤
   │ আপনি কি নিশ্চিত?                    │
   │ এই কাজটি পূর্বাবস্থায় ফেরানো যাবে না│
   ├─────────────────────────────────────┤
   │        [বাতিল] [মুছে ফেলুন]        │
   └─────────────────────────────────────┘
   ```
3. **[মুছে ফেলুন]** এ ক্লিক করুন
4. চুল্লির ধরন স্থায়ীভাবে মুছে যাবে

## সার্চ ও ফিল্টার ব্যবহার

### সার্চ বার

টেবিলের উপরে সার্চ বার থাকবে:

```
┌─────────────────────────────────────────────┐
│ 🔍 Search furnace types...                 │
└─────────────────────────────────────────────┘
```

### কী দিয়ে সার্চ করবেন?

সার্চ করলে নিচের ফিল্ডগুলোতে খুঁজবে:

1. **বাংলা নাম** (name_bn)
2. **ইংরেজি নাম** (name_en)
3. **শর্ট কোড** (short_code)

---

### সার্চ উদাহরণ

#### উদাহরণ ১: বাংলা নাম দিয়ে
```
সার্চ: হফম্যান

ফলাফল:
✅ হফম্যান (HOFF)
✅ হফম্যান টাইপ-১ (HOFF1)
❌ জিগজ্যাগ (ZIGZ)
```

#### উদাহরণ ২: ইংরেজি নাম দিয়ে
```
সার্চ: Tunnel

ফলাফল:
✅ Tunnel (TUNL)
❌ হফম্যান (HOFF)
```

#### উদাহরণ ৩: শর্ট কোড দিয়ে
```
সার্চ: HOFF

ফলাফল:
✅ হফম্যান (HOFF)
✅ হফম্যান টাইপ-১ (HOFF1)
❌ জিগজ্যাগ (ZIGZ)
```

---

### ফিল্টার ড্রপডাউন

সার্চ বারের পাশে ফিল্টার ড্রপডাউন থাকবে:

```
┌──────────────────┐
│ Status ▼         │
├──────────────────┤
│ All              │
│ Active           │
│ Inactive         │
└──────────────────┘
```

### ফিল্টার অপশন

| অপশন | কী দেখাবে |
|------|----------|
| **All** | সব চুল্লির ধরন (সক্রিয় + নিষ্ক্রিয়) |
| **Active** | শুধু সক্রিয় চুল্লির ধরন |
| **Inactive** | শুধু নিষ্ক্রিয় চুল্লির ধরন |

---

### ফিল্টার ব্যবহার উদাহরণ

#### শুধু সক্রিয় দেখতে:
1. **Status** ড্রপডাউনে ক্লিক করুন
2. **Active** সিলেক্ট করুন
3. শুধু সক্রিয় চুল্লির ধরন দেখাবে

#### শুধু নিষ্ক্রিয় দেখতে:
1. **Status** ড্রপডাউনে ক্লিক করুন
2. **Inactive** সিলেক্ট করুন
3. শুধু নিষ্ক্রিয় চুল্লির ধরন দেখাবে

#### আবার সব দেখতে:
1. **Status** ড্রপডাউনে ক্লিক করুন
2. **All** সিলেক্ট করুন
3. সব চুল্লির ধরন দেখাবে

---

### সার্চ + ফিল্টার একসাথে

আপনি সার্চ ও ফিল্টার একসাথে ব্যবহার করতে পারেন:

```
সার্চ: হফম্যান
ফিল্টার: Active

ফলাফল: শুধু সক্রিয় হফম্যান চুল্লির ধরন
```

---

### রিজাল্ট না পেলে

যদি সার্চ বা ফিল্টারের পর কোনো রেজাল্ট না পাওয়া যায়:

```
┌─────────────────────────────────────┐
│ 📭 No furnace types found          │
│                                     │
│ আপনার সার্চ বা ফিল্টারের সাথে       │
│ মিলে কোনো চুল্লির ধরন পাওয়া যায়নি │
└─────────────────────────────────────┘
```

**সমাধান**:
1. সার্চ টার্ম পরিবর্তন করুন
2. ফিল্টার রিসেট করুন (All সিলেক্ট করুন)
3. বানান চেক করুন

## সাধারণ সমস্যা ও সমাধান

### সমস্যা ১: "এই নামে ইতিমধ্যে চুল্লির ধরন আছে"

**এরর বার্তা**:
```
❌ This furnace type already exists.
```

**কারণ**: আপনি যে বাংলা নামটি দিচ্ছেন, সেটি ইতিমধ্যে ব্যবহৃত হয়েছে।

**সমাধান**:
1. ভিন্ন বাংলা নাম দিন
2. অথবা নামের সাথে সংখ্যা যোগ করুন:
   ```
   হফম্যান → হফম্যান টাইপ-১
   হফম্যান → হফম্যান-২
   ```

---

### সমস্যা ২: "এই কোডটি ইতিমধ্যে ব্যবহৃত হচ্ছে"

**এরর বার্তা**:
```
❌ This short code is already in use.
```

**কারণ**: আপনি যে শর্ট কোডটি দিচ্ছেন, সেটি ইতিমধ্যে আছে।

**সমাধান**:
1. ভিন্ন শর্ট কোড দিন
2. সংখ্যা যোগ করুন:
   ```
   HOFF → HOFF1
   HOFF → HOFF-2
   ```

---

### সমস্যা ৩: "শর্ট কোডে শুধু বড় অক্ষর ও সংখ্যা দেওয়া যাবে"

**এরর বার্তা**:
```
❌ Short code can only contain letters, numbers, dashes and underscores.
```

**কারণ**: শর্ট কোডে ছোট হাতের অক্ষর বা বিশেষ অক্ষর ব্যবহার করেছেন।

**সমাধান**:
1. শুধু বড় হাতের অক্ষর (A-Z) ব্যবহার করুন
2. সংখ্যা (0-9) ব্যবহার করতে পারেন
3. ড্যাশ (-) বা আন্ডারস্কোর (_) ব্যবহার করতে পারেন
4. ছোট অক্ষর লিখলে অটোমেটিক বড় অক্ষরে পরিবর্তন হবে

**গ্রহণযোগ্য**:
```
✅ HOFF
✅ HOFF1
✅ HOFF-1
✅ HOFF_1
```

**অগ্রহণযোগ্য**:
```
❌ hoff (ছোট অক্ষর)
❌ Hoff (মিশ্রিত)
❌ H@FF (বিশেষ অক্ষর)
❌ HOFF.1 (ডট ব্যবহার)
```

---

### সমস্যা ৪: ডায়ালগ বক্স বন্ধ হচ্ছে না

**লক্ষণ**: তৈরি বা আপডেট করার পর ডায়ালগ বক্স বন্ধ হচ্ছে না।

**সমাধান**:
1. পেজ রিফ্রেশ করুন (F5 চাপুন)
2. ব্রাউজার ক্যাশে ক্লিয়ার করুন
3. ইন্টারনেট সংযোগ চেক করুন

---

### সমস্যা ৫: তালিকা লোড হচ্ছে না

**লক্ষণ**: চুল্লির ধরন তালিকা দেখা যাচ্ছে না।

**সমাধান**:
1. পেজ রিফ্রেশ করুন
2. ব্রাউজার পরিবর্তন করে দেখুন (Chrome/Firefox)
3. ইন্টারনেট সংযোগ চেক করুন
4. সমস্যা চললে অ্যাডমিনকে জানান

## সতর্কতা ও পরামর্শ

### ⚠️ সতর্কতাসমূহ

#### ১. চুল্লির ধরন তৈরির আগে ভাবুন
- নতুন ধরন তৈরি করার আগে ভাবুন এটি আসলে প্রয়োজন কিনা
- বিদ্যমান ধরন ব্যবহার করা যায় কিনা দেখুন

#### ২. শর্ট কোড সাবধানে দিন
- শর্ট কোড পরে পরিবর্তন করা যাবে না (রিপোর্টে সমস্যা হবে)
- সহজ ও মনে রাখার মতো কোড দিন

#### ৩. নিষ্ক্রিয় করার আগে চেক করুন
- নিষ্ক্রিয় করার আগে দেখুন এই ধরনে চুল্লি আছে কিনা
- থাকলে সেগুলো অন্য ধরনে স্থানান্তর করুন

#### ৪. নামে সংখ্যা ব্যবহার
- একই ধরনের একাধিক ভেরিয়েশন থাকলে নামে সংখ্যা যোগ করুন
  ```
  হফম্যান-১, হফম্যান-২
  ```

| আধুনিক/অটোমেটেড | ⚙️ Cog |
| দ্রুত উৎপাদন | ⚡ Zap |
| বড় কারখানা | 🏭 Factory |

#### ৩. বিবরণ যোগ করুন
সম্ভব হলে বিবরণ যোগ করুন:
- চুল্লির বৈশিষ্ট্য
- উৎপাদন ক্ষমতা
- বিশেষ নির্দেশনা

#### ৪. নিয়মিত রিভিউ
প্রতি মাসে চুল্লির ধরন রিভিউ করুন:

- কোন ধরনগুলো নিষ্ক্রিয় করা যায়
- নতুন ধরন প্রয়োজন কিনা
