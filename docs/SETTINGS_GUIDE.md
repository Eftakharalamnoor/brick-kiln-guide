# ⚙️ Documentation Settings - সম্পূর্ণ গাইড

## 📋 সেটিংস পেজের বৈশিষ্ট্যসমূহ

### ১. **ফন্ট সেটিংস (Font Settings)** 🔤

#### যা যা আছে:
- ✅ **ফন্ট সাইজ** - ছোট (১২px), মাঝারি (১৪px), বড় (১৬px), এক্সট্রা বড় (১৮px)
- ✅ **ফন্ট ফ্যামিলি** - হিন্দ শিলিগুড়ি, সোলAIMAN লিপি, নিরালি
- ✅ **লাইন হাইট** - ১.০ থেকে ২.০ পর্যন্ত স্লাইডার
- ✅ **লাইভ প্রিভিউ** - পরিবর্তন সাথে সাথে দেখা যাবে

#### ব্যবহার:
```
1. সেটিংস → ফন্ট সেটিংস
2. ফন্ট সাইজ নির্বাচন করুন
3. ফন্ট ফ্যামিলি নির্বাচন করুন
4. লাইন হাইট এডজাস্ট করুন
5. "সেভ করুন" এ ক্লিক করুন
```

### ২. **রঙ থিম (Color Theme)** 🎨

#### যা যা আছে:
- ✅ **থিম মোড** - লাইট, ডার্ক, হাই কন্ট্রাস্ট
- ✅ **প্রাইমারি কালার** - ৬টি প্রি-ডিফাইন্ড কালার
- ✅ **অটোমেটিক সেভ** - পছন্দের থিম সেভ থাকে

#### থিম সমূহ:
| থিম | আইকন | ব্যবহার |
|------|------|---------|
| লাইট | ☀️ | দিনের বেলা, স্বাভাবিক ব্যবহার |
| ডার্ক | 🌙 | রাতের বেলা, চোখের আরাম |
| হাই কন্ট্রাস্ট | 👁️ | চোখের সমস্যা, বিশেষ প্রয়োজন |

#### প্রাইমারি কালার:
- 🟠 কমলা (ডিফল্ট)
- 🔵 নীল
- 🟢 সবুজ
- 🔴 লাল
- 🔵 গাঢ় নীল
- 🟣 বেগুনি

### ৩. **আইকন সেটিংস (Icon Settings)** 🎯

#### যা যা আছে:
- ✅ **পেজ আইকন** - সাইডবার মেনুতে আইকন দেখানো/লুকানো
- ✅ **সেকশন আইকন** - সেকশন হেডারে আইকন দেখানো/লুকানো
- ✅ **স্ট্যাট কার্ড আইকন** - পরিসংখ্যান কার্ডে আইকন দেখানো/লুকানো
- ✅ **সার্চ আইকন** - সার্চ বারে আইকন দেখানো/লুকানো

#### টগল কন্ট্রোল:
```
প্রতিটি অপশনের পাশে টগল সুইচ আছে:
- চালু (নীল) = আইকন দেখাবে
- বন্ধ (ধূসর) = আইকন লুকানো
```

---

### ৪. **লেআউট সেটিংস (Layout Settings)** 📐

#### যা যা আছে:
- ✅ **সাইডবার প্রস্থ** - ২০০px থেকে ৪০০px পর্যন্ত
- ✅ **কন্টেন্ট প্রস্থ** - ফুল উইথ / কনস্ট্রেইন্ড (১২০০px)
- ✅ **ডান প্যানেল** - সেকশন লিঙ্ক প্যানেল দেখানো/লুকানো

#### সাইডবার প্রস্থ:
| সাইজ | প্রস্থ | ব্যবহার |
|------|------|---------|
| সংকীর্ণ | ২০০px | ছোট স্ক্রিন, কম জায়গা |
| মাঝারি | ২৬০px | স্বাভাবিক ব্যবহার (ডিফল্ট) |
| প্রশস্ত | ৪০০px | বড় স্ক্রিন, বেশি মেনু |

---

### ৫. **ভাষা সেটিংস (Language Settings)** 🌐

#### যা যা আছে:
- ✅ **বাংলা** 🇧🇩 - বর্তমান ভাষা
- ✅ **English** 🇬🇧 - ইংরেজি ভাষা

#### ভাষা পরিবর্তন:
```
1. সেটিংস → ভাষা সেটিংস
2. বাংলা বা English নির্বাচন করুন
3. "সেভ করুন" এ ক্লিক করুন
4. সব পেজ নির্বাচিত ভাষায় দেখাবে
```

---

### ৬. **প্রিন্ট সেটিংস (Print Settings)** 🖨️

#### যা যা আছে:
- ✅ **ব্যাকগ্রাউন্ড কালার** - প্রিন্টে রঙিন ব্যাকগ্রাউন্ড অন্তর্ভুক্ত
- ✅ **লিঙ্ক URL** - প্রিন্টে লিঙ্কের URL দেখাবে
- ✅ **টেবিল অফ কন্টেন্ট** - প্রিন্টের শুরুতে বিষয়সূচি
- ✅ **পেজ নম্বর** - প্রিন্টে পেজ নম্বর যোগ

#### প্রিন্ট অপ্টিমাইজেশন:
- স্বয়ংক্রিয়ভাবে প্রিন্ট-ফ্রেন্ডলি ফরম্যাট
- পৃষ্ঠা সংখ্যা কমানো
- কালি সাশ্রয়ী মোড

---

### ৭. **ডিফল্টে রিসেট (Reset to Defaults)** 🔄

#### যা যা আছে:
- ✅ **সব সেটিংস রিসেট** - এক ক্লিকে সব ডিফল্টে ফিরিয়ে আনা
- ✅ **কনফার্মেশন ডায়ালগ** - ভুলবশত রিসেট থেকে রক্ষা
- ✅ **সতর্কবার্তা** - কী কী মুছে যাবে তা দেখানো

#### সতর্কতা:
```
⚠️ রিসেট করলে যা মুছে যাবে:
- ফন্ট সাইজ সেটিংস
- রঙ থিম পছন্দ
- আইকন ভিজিবিলিটি
- লেআউট সেটিংস
- ভাষা পছন্দ
- প্রিন্ট সেটিংস
```

---

### ৮. **সেটিংস সেভ ও ম্যানেজমেন্ট** 💾

#### যা যা আছে:
- ✅ **অটো সেভ** - সব সেটিংস স্বয়ংক্রিয়ভাবে সেভ
- ✅ **ম্যানুয়াল সেভ** - "সেভ করুন" বাটন
- ✅ **এক্সপোর্ট** - সেটিংস ফাইল ডাউনলোড (.json)
- ✅ **ইম্পোর্ট** - সেটিংস ফাইল আপলোড
- ✅ **প্রয়োগ করুন** - পরিবর্তন সাথে সাথে প্রয়োগ

#### সেটিংস স্টোরেজ:
```
সেটিংস সেভ থাকে:
- ব্রাউজারের LocalStorage এ
- ডিভাইস স্পেসিফিক
- ব্রাউজার স্পেসিফিক
```

## 🎯 সেটিংস ব্যবহারের গাইড

### প্রথমবার সেটআপ:

```
১. সেটিংস পেজে যান (মেনু → সেটিংস → ডকুমেন্টেশন সেটিংস)
২. ফন্ট সাইজ সেট করুন (মাঝারি recommended)
৩. রঙ থিম নির্বাচন করুন (লাইট/ডার্ক)
৪. আইকন সেটিংস ঠিক করুন
৫. লেআউট পছন্দ সেট করুন
৬. ভাষা নির্বাচন করুন
৭. "প্রয়োগ করুন" এ ক্লিক করুন
```

### দৈনন্দিন ব্যবহার:

```
সাধারণত একবার সেট করলে আর পরিবর্তন করতে হয় না।
প্রয়োজন হলে যেকোনো সময় পরিবর্তন করা যাবে।
```
